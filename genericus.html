How It Works

From idea to running system in minutes

No code changes, no redeployments — just configuration.

Model Your Domain

Use the Admin UI or CLI to define entities (e.g., Company, Employee, Asset), their fields, data types, and validation rules.

Define Relationships

Connect entities with 1:1, 1:N, or N:N relationships. Give each direction a friendly name — Genericus handles the rest.

Publish Metadata

Validate your schema and publish it. Genericus instantly activates the new configuration across all API endpoints and generated UIs.

Use Your System

End users interact with auto-generated forms and list views. Developers consume the versioned REST API or generated OpenAPI client.
